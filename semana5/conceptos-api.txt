La ruta POST /reporte recibe información enviada por el usuario mediante una solicitud HTTP POST. En este caso, recibe un mensaje desde el cuerpo de la solicitud (req.body.mensaje) y devuelve una respuesta en formato JSON confirmando que el reporte fue recibido.

Esta ruta podría servir en una aplicación comunitaria para registrar reportes ciudadanos, alertas de seguridad, daños en infraestructura, solicitudes de apoyo o cualquier información que la comunidad necesite compartir de manera organizada. De esta forma, los datos pueden almacenarse, procesarse y consultarse posteriormente.
