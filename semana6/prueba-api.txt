Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

El servidor recibe los datos mediante req.body y devuelve una respuesta en formato JSON con la información recibida:

{
"estado": "Datos recibidos",
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

Esto confirma que la aplicación está procesando correctamente los datos enviados por el usuario.
