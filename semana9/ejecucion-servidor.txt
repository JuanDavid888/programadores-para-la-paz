Al ejecutar el comando node server.js, el servidor se inicia y comienza a escuchar solicitudes en el puerto 3000. En la terminal aparece el mensaje “Servidor ejecutándose en puerto 3000”.

Cuando se visita la ruta http://localhost:3000, el servidor responde con el mensaje “Servidor activo y funcionando”.

En la ruta /estado, el servidor devuelve una respuesta en formato JSON que indica el estado del sistema y el nombre del servicio.

Mensaje de respuesta:

{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}
